# Taller de Recursividad en Java 🚀

Este repositorio contiene la solución completa a un taller de **Programación Orientada a Objetos** enfocado en **Algoritmos Recursivos**. Cada ejercicio está estructurado de forma independiente siguiendo principios de limpieza de código y separación de responsabilidades.

## 📋 Contenido del Taller

El taller abarca desde operaciones aritméticas básicas hasta funciones de complejidad avanzada:

1.  **Factorial:** Cálculo del factorial de un número.
2.  **Inversión de Cifras:** Invertir un número entero matemáticamente.
3.  **Serie Armónica:** Sumatoria de $1 + 1/2 + 1/3 + ... + 1/n$.
4.  **Suma de Dígitos:** Sumar cada dígito de un número entero.
5.  **Sumatoria Simple:** Suma lineal de $1$ hasta $n$.
6.  **Potenciación:** Cálculo de $a^b$ mediante multiplicaciones.
7.  **M.C.D. (Euclides):** Máximo Común Divisor mediante el algoritmo clásico.
8.  **Copia de Cadenas:** Duplicación de Strings carácter por carácter.
9.  **División Entera:** Cálculo del cociente mediante restas sucesivas.
10. **Multiplicación:** Producto de dos números mediante sumas sucesivas.
11. **Suma de Vectores:** Sumatoria de los elementos de un arreglo unidimensional.
12. **Suma de Matrices:** Procesamiento de arreglos bidimensionales ($m \times n$).
13. **Fibonacci:** Generación de la serie hasta un límite definido.
14. **Función de Ackermann:** Implementación de recursividad profunda y anidada.

## 🛠️ Estructura del Proyecto

Cada punto del taller se divide en tres componentes clave para mantener el orden y evitar conflictos de nombres:

* **Lógica (`NombreMotor.java`):** Contiene únicamente los métodos recursivos puros.
* **Gestión (`NombreControlador.java`):** Maneja la interacción con el usuario (`Scanner`) y la validación de datos.
* **Ejecución (`MainTema.java`):** Punto de entrada principal para ejecutar el ejercicio específico.

> [!IMPORTANT]
> Los archivos están diseñados para ejecutarse de forma independiente. No se requiere configuración de `packages`, facilitando la portabilidad en cualquier IDE (VS Code, IntelliJ, Eclipse).

## 🚀 Cómo ejecutar

1. Clona este repositorio o descarga los archivos.
2. Abre la carpeta en tu editor preferido (recomendado **VS Code**).
3. Busca el archivo `Main` del ejercicio que deseas probar (ejemplo: `MainAckermannTotal.java`).
4. Ejecuta el archivo y sigue las instrucciones en la terminal.
